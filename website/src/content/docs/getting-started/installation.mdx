## Rust

```bash
cargo add blazen
```

To enable specific provider features:

```bash
cargo add blazen --features openai,anthropic
```

## Python

Preferred (using [uv](https://docs.astral.sh/uv/)):
```bash
uv add blazen
```

Or with pip:
```bash
pip install blazen
```

## Node.js

```bash
pnpm add blazen
```

Or with npm/yarn:

```bash
npm install blazen
yarn add blazen
```

## CLI

```bash
cargo install blazen-cli
```
